Java面向对象

程序有的是面向过程的，有的是面向对象，面向过程而言从上到下一步步的走代码复用率低，而面向对象既方便代码也节省因此更实用。

面向过程：强调的是功能行为,一种过程,先干啥,再干啥;

面向对象：将功能封装到对象里，强调的是具备某功能的对象

面向对象的三个特征：

  • 封装
  • 继承
  • 多态

类与对象之间的关系：

类是对生活中某一类事物的描述（一种抽象概念），对象是该类事物实实在在存在的个体。(对象==实例)

比如说，手机就是一个类，手机有很多种，iPhone，HTC……这么多品牌都是手机，因为他们都具备了手机的基本功能----通讯。
因此他们每一个个例就是一个对象，他们属于一类--手机。

类的定义：

[修饰符] class 类名{
1~n个构造方法；
0~n个字段；
0~n个方法
}

构造方法：用来构造类的实例（默认有一个无参的构造方法，得使用new调用）
字段：类或对象所包含的数据，对类状态的一种描述；
方法：类或对象的特征或行为

对象的创建，定义，以及使用：

类名 对象名称 = new 类名（）；
这样就创减了一个对象。

例子：Phone phone = new Phone（）；

匿名对象：

对方法或字段只进行一次调用，没有具体的名称只能使用一次。
例子：new Phone（）；
调用方法 new Phone.show（）；调用了Phone类中的show方法；

可作为实际参数进行传递
例子：
public void show(Car c){ }
new Car().show(new Car());

只在堆内存中开辟空间，而没有在栈内存的引用。
